<Step title="Extract">
Extract with [7-Zip](https://7-zip.org/)
</Step>
<Step title="Run">
Use `run_nvidia_gpu` to run it using your NVIDIA GPU.

Use `run_cpu` to run it using only your CPU.
</Step>
